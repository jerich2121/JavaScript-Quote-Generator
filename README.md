# JavaScript-Quote-Generator
A JavaScript Quote Generator project built using DOM Manipulation. Users can generate random quotes at the click of a button. This project demonstrates core JavaScript concepts such as event listeners, DOM selection, and dynamic content updates, along with a clean and responsive UI built with HTML and CSS.
